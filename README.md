# 🌐SwiftBites
SwiftBites is a food delivery system built to help small restaurants run their own online ordering platform without relying on third-party services.

> [!NOTE]
> Project is under development, new features will be added continuously.

<br>

## 💠Tech Stack
▫️<b>FrontEnd : </b>React, BootStrap, Axios, Vite<br>
▫️<b>BackEnd : </b>Java 17, Spring Boot<br>
▫️<b>APIs : </b>RESTful APIs, PostMan<br>
▫️<b>Dependencies : </b>Lombok, Spring Validation, Spring Boot DevTools<br>
▫️<b>Others : </b>H2 Database (Local DataBase), Maven (Build Tool)<br>


<br>

## 💠Features

### ▫️User Experience
> Browse food menu, filter by category, search items, view food details, manage cart, place orders, and track order status <I>(preparing, out for delivery, delivered) </I>. Secure authentication with signup/login, billing address support, and order history.

### ▫️Payments & Invoicing
> Order placement with billing details and order tracking.

### ▫️Admin Panel
> Admin authentication with dashboard to add/manage food items and monitor orders.

### ▫️Additional
> Contact support via email, social links.

<br>

## 💠Run Project Locally

```bash
git clone https://github.com/divyanshsood22/swiftbites.git
cd swiftbites-api
./mvnw clean install
./mvnw spring-boot:run
```
> [!IMPORTANT]
> Create a `.env` file in the root directory. Refer to `.env.example` for required variables.
